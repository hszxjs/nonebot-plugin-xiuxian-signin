# nonebot-plugin-xiuxian-signin

一个面向 OneBot V11 的 NoneBot2 修仙签到插件。所有对外回复都以图片面板发送，包含签到、垂钓、背包、突破、历练、特殊能力、秘境和榜单，并支持 PicMenu Next 菜单读取。

## 功能

- 直接发送 `签到` 即可每日签到。
- 发送 `帮助` 可查看修为提升路径和常用入口，发送 `图鉴` 可查看境界、突破、丹药、符箓、武器、功法、材料和特殊能力资料。
- 发送 `面板` 可直接打开个人修为面板，旧的 `我的修为`、`境界`、`灵根` 仍可使用。
- 首次签到会抽取个人灵根。
- 灵根由品阶、品质、属性组成：
  - 品阶：天阶、地阶、玄阶、黄阶、凡品
  - 品质：下品、中品、上品、极品
  - 属性：金、木、水、火、土、雷、冰
- 灵根属性会影响图片主色，灵根品阶和品质会影响升级经验长度与每日修炼涨幅。
- 签到图包含 QQ 头像、灵根、签到次数、累计修为、境界和经验进度。
- 插件内置 `NotoSansSC-VF.ttf` 中文字体，个人信息卡使用修仙主题背景和更大字号。
- 每次成功签到获得 1 次诸天万界垂钓机会。
- 垂钓次数可累积，支持单次垂钓和 10 连抽。
- 处于境界瓶颈时，签到/垂钓触发突破材料机缘会提高当前突破道具权重；一般 10 次签到内较容易拿到可用道具，但道具名称决定品相上限，品阶和品质只在该上限内提升结果。
- 垂钓、秘境探索和境界突破余韵均有更低概率掉落 `九秘残页`、`八禁感悟`、`九禁感悟`、`神禁烙印` 等特殊能力传承材料，玩家可通过 `特殊能力图鉴` 明确追求路径。
- 境界圆满进入瓶颈后，修为不再直接增长；溢出修为会按 50% 凝成精纯灵液，突破后不会清空灵液，可发送 `炼化灵液` 将灵液转回自身修为。
- 垂钓奖池覆盖 14 类物品、5 个品阶，已扩展至 400+ 奖励；灵器和功法按七系灵根扩充，仙缘与阵盘保留高数量池。
- 功法和灵器会带有需求灵根，宿主拥有对应主灵根、额外灵根或已炼成的后天灵根时显示契合。
- 新增后天灵根：妖丹可炼成丹灵根，适配五行的灵器可祭炼成器灵根，用于化神突破炼虚时的五行补全。
- 群聊会按每天发言次数排名，每晚 22:00 自动发布话痨榜并发放奖励，并同步发布群修为榜与群战力榜。
- 新增历练板块：可查看灵器、装备灵器、计算战力、群内 PK，并生成战力榜。
- 所有机器人回复均输出为图片面板：包括错误提示、榜单、背包、突破、秘境、道具使用和历练切磋。
- 新增趣味板块 `天机占卜`：每天每个群需要先由第一位签到的天机阁门人坐堂；其他天机阁门人可发送 `坐堂` 加入分润，有坐堂后才能问卦，坐堂者按占卜人数平分灵石润资。
- 新增趣味板块 `斗地主`：支持三人桌、人机局、修为威压抢地主、原地主保留后强制普通斗法、加倍、春天/反春天；炸弹在牌局中显示为雷劫，王炸显示为天罚雷劫。
- 通用面板为各功能和道具类型内置图标：签到、帮助、图鉴、灵根、突破、垂钓、榜单、灵器、功法、阵盘、傀儡、灵植、丹药、符箓、灵石、精纯灵液、灵食、奇物、杂物、背包、秘境、天机占卜、斗地主扑克、特殊能力/神通和警示。
- 新增境界突破、背包道具使用、傀儡、灵植和 60 秒限时秘境入口。
- 每日签到时有低概率触发奇遇，改善资质或觉醒额外灵根。
- 新增修炼路线：剑修、术修、炼丹师、阵法师、炼器师；邪修可与主路线同修。
- 新增身份令牌：天机阁与合欢宗路线身份会在个人面板展示，并提供天机秘境提示与双修次数。
- 新增每日任务：每天签到后自动生成 5 个任务，可完成获取修为、灵石与少量垂钓次数。
- 新增商店、万宝楼与货币系统：每日全服共通刷新 8 个商品，所有垂钓物品都有灵石价格，支持购买、单件出售、批量出售、指定交易和万宝楼公开寄售。
- 灵石按 100 下品 = 1 中品、100 中品 = 1 上品、100 上品 = 1 极品自动折算显示。
- 新增炼丹：炼丹师可消耗灵材、灵植与灵石炼制筑基丹、小还丹、大还丹、元婴丹等丹药；材料品阶和品质会影响成功率、升品率和最终丹药品质。
- 新增炼器：炼器师可消耗灵材和灵石炼制灵器、阵盘和仿制帝兵；支持青竹蜂云剑、庚金青竹蜂云剑、大庚剑阵等图谱，也可祭炼本命灵器。
- 新增符箓绘制：可消耗灵石绘制现有全部符箓；突破用符、符诏、法旨等需达到对应突破前境界巅峰才能绘制。
- 新增散功重修：可回退至上一境界后期，移除当前境界品相，重修后重新突破以改善品质。
- 秘境入口超时会在 60 秒后主动发送图片提示；天机阁特殊秘境会提示坏结局选项。六类普通秘境各有 100 个随机探索事件，并额外抽取 20 类高危险地入口；高危险地 5 选 2 生路，无首领挑战，奖励更偏向唯一道具、仙种、帝兵与高阶灵材。
- 突破方式扩展为丹破、悟道、天命、祭炼、斩执等类型；突破文案按境界生成金丹异象、元婴天兆、化神问心、雷劫和仙门等氛围描述。
- 图标改为更具象的图形：灵器为剑、功法为书、杂物为扫帚，并新增任务、商店、炼丹、身份令牌图标。

## 斗地主

斗地主是群聊趣味牌局，所有提示都使用图片面板输出。手牌会尝试私聊发送，群内可发送 `手牌` 重新查看。

- 炸弹不再直接显示为“炸弹”，而是按牌面显示为 `3重雷劫`、`A重雷劫` 等。
- 王炸显示为 `天罚雷劫`，可用 `出牌 小王大王`、`出牌 王炸` 或 `出牌 天罚雷劫` 触发。
- 雷劫和天罚雷劫都会让当前倍数翻倍。
- 春天和反春天会在结算时自动判定，达成后再翻倍。

## 每日话痨榜

插件会自动统计群聊普通发言数量。每天 22:00 后，每个群会独立结算一次当日排行榜。

奖励规则：

| 排名 | 奖励 |
| ---: | --- |
| 第 1 名 | +36 修为，+10 次垂钓 |
| 第 2 名 | +28 修为，+8 次垂钓 |
| 第 3 名 | +22 修为，+6 次垂钓 |
| 第 4 名 | +16 修为，+4 次垂钓 |
| 第 5 名 | +16 修为，+2 次垂钓 |
| 第 6-10 名 | +10 修为 |

如果用户尚未签到、还没有灵根，修为奖励会先进入 `待领取修为`，首次签到时自动汇入经验进度。

话痨榜奖励发放完成后，插件会在同一条结算消息中追加：

- `群修为榜`：按本群成员总修为与待领取修为排序。
- `群战力榜`：按境界、修为、灵根和已装备灵器计算后的战力排序。

## 突破道具评级

突破道具现在分为“道具名上限”和“品阶/品质”两层判定。瓶颈期签到仍有 50% 概率额外获得当前突破道具，因此不会把获取难度拉得太高；但基础道具更常见，高品阶基础道具也只能达到自己的品相上限。

- 筑基示例：`筑基丹` 上限较低，`地脉筑基液` 可到中高品相，`天道筑基露` 才有机会冲击天道筑基。
- 金丹示例：`小还丹`、`大还丹` 偏基础，`凝魄金丹`、`造化金丹` 上限更高；天阶极品小还丹不会直接结成一品金丹。
- 高阶示例：化神后可能出现仙阶同名突破道具，但仍受道具名上限约束；想冲最高品相，需要对应高上限道具本身也达到足够品阶和品质。

发送 `突破` 会显示当前瓶颈所需道具和各自品相上限；发送 `品相图鉴` 可查看全部突破道具与品相上限对应关系；发送 `图鉴 破虚灵引` 可查看单个道具的故事、获取方式、制作素材和品相上限。

## 特殊能力与神通

特殊能力是斗法和长期追求线的一部分，不会再由斗法资料初始化自动赠送。玩家需要先获得传承材料，再主动参悟。

获取路径：

- `垂钓`：特殊能力已加入垂钓奖池，且每次垂钓有额外灵光概率掉落传承材料。
- `秘境`：上古宗门遗址、兽潮、上古大能洞府、太古矿区、虚神界残域和青铜仙殿都可能产出传承材料，其中虚神界残域、青铜仙殿和洞府权重更高。
- `秘境首领挑战`：进入秘境后会出现首领挑战选项，成功可直接获得 10 次探索能得到的随机奖励，并额外获得该首领妖丹，同时提升当前功法熟练度和阵法熟练度；失败进入 5 分钟反噬禁修期。
- `突破`：境界突破成功后有概率触发突破余韵，额外落下一份传承材料。

使用流程：

1. 发送 `特殊能力图鉴` 查看完整图鉴、材料来源和斗法加成。
2. 获得材料后发送 `背包` 或 `特殊能力` 查看可领悟材料编号。
3. 发送 `领悟特殊能力 1` 或 `领悟神通 1` 消耗对应材料。
4. 普通斗法中直接发送能力名或别名，例如 `开启八禁`、`斗字秘`、`神禁领域`，即可尝试触发。

核心材料包括：`九秘残页`、`八禁感悟`、`九禁感悟`、`神禁烙印`、`掌天瓶影拓片`、`斗战圣法残卷`、`至尊骨符文`、`他化自在影`、`鲲鹏极速羽痕` 等。

当前图鉴收录 33 种特殊能力：八禁、九禁、神禁领域、九秘九式、掌天瓶影、青元剑芒、元婴天兆、道经轮海、斗战圣法、荒古圣血、青帝莲相、至尊骨符文、以身为种、他化自在影、十洞天神环、鲲鹏极速、柳神法相、古神一指、岁月意境、杀戮本源、梦道轮回、因果断线、牧天九歌、天问一式、神藏开阖。
固定品阶：所有特殊能力均已固定品阶与品质，图鉴会逐项显示；`八禁`=玄阶极品，`九禁`=地阶中品，`神禁领域`=天阶下品，`九秘九式`=天阶极品。八禁 -> 九禁 -> 神禁为进阶路线，升级后只保留最高级能力；九秘获得 n 件后，已获九秘效果按 n 倍增强。

## 修炼路线、身份与货币

- `剑修`：装备剑类灵器时，战力按伤害提升 30% 计入。
- `术修`：装备非剑类灵器时，法术伤害按 30% 加成计入。
- `炼丹师`：可使用灵材、灵植与灵石炼制丹药，新增材料已进入垂钓与秘境奖励；当前灵材奖池均至少拥有一条炼丹或炼器用途，可发送 `图鉴 灵材名` 反查配方用途。
- `阵法师`：阵法熟练度提升更快，阵法效果额外增强。
- `炼器师`：可使用灵材和灵石炼制灵器、傀儡、阵盘与仿制帝兵，并能围绕本命灵器成长。
- `邪修`：可与主路线同修，能识破邪修陷阱；当前秘境坏结局/首领挑战失败的反噬禁修期统一缩短为 5 分钟。
- `天机阁` 身份：发送 `选择身份 天机阁弟子`、`选择身份 天机阁长老`、`选择身份 天机阁太上长老`。弟子需筑基；长老需元婴且弟子身份签到 10 天；太上长老需炼虚且长老身份签到 30 天。身份效果为特殊秘境示警，弟子 7 天一次，长老 5 天一次，太上长老每天一次；每天本群第一位成功签到的天机阁门人会坐堂，其他天机阁门人可发送 `坐堂` 加入分润，占卜润资将按次平分。
- `合欢宗` 身份：发送 `选择身份 合欢宗弟子`、`选择身份 合欢宗长老`、`选择身份 合欢宗太上长老`。弟子需练气中期；长老需金丹且弟子身份签到 10 天；太上长老需化神且长老身份签到 20 天。身份效果为每日 1/2/5 次双修。
- 灵石为基础货币，内部以“下品灵石”为单位存储，面板会自动折算为中品、上品、极品灵石。

## 后天灵根与五行补全

化神突破炼虚需要五行合一，系统现在只认可先天灵根、额外灵根和已经炼成的后天灵根；单纯持有妖丹、灵器或奇物不会直接补全五行。

- `丹灵根`：消耗灵材栏中的妖丹炼成，属性来自妖丹本身或妖力倾向，精纯度受妖兽生前修为、妖丹品阶和品质影响；后天性比先天性弱，最高只与地灵根持平。
- `器灵根`：用适配五行属性的灵器祭炼，成功率为 35%；成功后不消耗灵器，但灵根依托该灵器存在，灵器出售或损毁后对应器灵根会失效。
- 查看入口：发送 `后天灵根` 或 `五行补全`。
- 炼化入口：发送 `炼化丹灵根 1`、`炼化器灵根 1`，编号分别来自背包灵材栏和灵器列表。

## 历练、灵器与战力

灵器主要来自诸天万界垂钓、秘境和商店。垂钓到的 `灵器` 可通过 `我的灵器` 查看，并使用 `装备灵器 编号 主手/副手/护甲` 装备到三类槽位，每个槽位同一时间只生效一件。

- 灵器拥有自己的品阶与品质，品阶越高战力加成越高，并会根据最低使用修为提升最终战力评价。
- 灵器装备会检查最低修为限制，修为不足时无法驾驭。
- `祭炼本命灵器 1` 可指定一件灵器随境界成长，并额外计入战力。
- `仙种` 为真仙后装备栏，可通过高危险地获得；`以身为种仙种` 等部分仙种具有全局唯一性。
- 极道帝兵为全局唯一道具，本体被某位玩家获得后，后续同名奖励会转为仿制品；发送 `帝兵图鉴` 可查看炼制者、事迹、专属技能和当前拥有者。
- 部分灵器会要求对应属性灵根；没有对应主灵根、额外灵根或后天灵根时无法装备。
- 战力由境界、累计修为、灵根资质、额外/后天灵根、签到积累、三槽灵器、符箓栏、功法、阵盘和傀儡共同计算。
- 群内发送 `pk @群友` 可以进行切磋。切磋只做胜负推演，不消耗修为，也不会损毁灵器。
- 群内发送 `战力榜` 可查看本群战力前十。

## 灵材用途与批量使用

- 所有垂钓灵材都已接入炼丹或炼器配方，不再存在只可获得但无用途的灵材。
- 发送 `图鉴 灵材名` 可以查看简介、获取方式和“配方用途”，例如 `图鉴 兽巢骨粉`。
- 支持 `批量使用丹药 10`、`批量服用丹药 全部`、`批量炼化灵石 20`、`炼化灵石 全部`、`批量使用灵食 10`。批量服丹会保护所有突破道具，避免误吃筑基丹、小还丹、造化金丹等关键材料。

## 炼器、帝兵与交易

- `炼器` 会打开可炼制图谱面板，炼制需选择 `炼器师` 路线。
- `炼器 青竹蜂云剑` 按图谱消耗灵材和灵石，材料品质越高，越容易得到更高品质成品。
- `青竹蜂云剑`、`庚金青竹蜂云剑` 与 `大庚剑阵` 已加入炼器与秘境材料线；青竹蜂云剑具有辟邪神雷设定，对邪修更克制。
- `阵法推演 1` 会消耗 2 件同名同阶同品阵盘提升品质，极品后升至上一品阶下品。
- `万宝楼挂售 灵器 1` 可将背包物品公开寄售，价格自动为系统回收价的 1.5 倍；`万宝楼购买 1` 购买，`万宝楼下架 1` 撤回。
- `交易 @群友 灵器 1 100` 保留为指定玩家交易；不 @ 或不带 QQ 时视为全群可接单。
- `接受交易 1` 会先检查买家灵石，不足时不会锁定挂单；`取消交易 1` 可撤回本人挂单。
- `批量出售 杂物 20` 适合清理背包，唯一道具和帝兵本体不会被批量出售。

## 奇遇与资质

每日签到时会判定一次奇遇：

- 未达到 `天阶极品` 前，每天有 `1/365` 概率经历奇遇。
- 奇遇出现后有 `50%` 概率提升一档资质。
- 资质提升顺序优先提升品质：`下品 -> 中品 -> 上品 -> 极品`。
- 如果已经是某一品阶的极品，则提升到上一品阶下品，例如 `黄阶极品 -> 玄阶下品`。
- 达到 `天阶极品` 后，不再提升主灵根资质。
- 达到 `天阶极品` 后，每天改为以 `(1+n)/999` 概率觉醒一条随机额外灵根，`n` 是当前额外灵根数量。
- 额外灵根会影响功法和灵器的契合判定。

## 全图片面板 UI

插件所有对外回复都使用图片输出：签到和垂钓使用专属卡片，其他功能使用统一的修仙主题面板。通用面板支持标题、副标题、分区、条目和底部提示，并根据玩家灵根颜色调整强调色。

内置图标包括：签到、灵根/境界、突破、垂钓、榜单、战力、切磋、灵器、功法、阵盘、傀儡、灵植、丹药、符箓、灵石、灵食、奇物、杂物、背包、秘境、天机占卜、斗地主扑克、警示。图标由 Pillow 绘制，无需额外素材文件。

## PicMenu Next 支持

插件已在 `PluginMetadata.extra` 中写入 PicMenu Next 可读取的菜单数据：

- `extra["pmn"]["markdown"] = True`
- `extra["funcs"] = [...]`
- `extra["menu_data"] = [...]`

其中 `funcs` 使用 PicMenu Next 外部菜单兼容的功能项字段：

- `func`
- `trigger_method`
- `trigger_condition`
- `brief_des`
- `detail_des`

当前 PicMenu Next 菜单已整理为 10 个直观分组：开始修仙、背包与图鉴、突破与修为、历练与装备、秘境与任务、路线与身份、炼丹炼器与符箓、交易与资源处理、趣味玩法、排行。
`突破与修为` 分组包含 `后天灵根`，用于查看丹灵根/器灵根与五行补全；`历练与装备` 分组包含 `炼化器灵根 1`，方便从灵器玩法进入。
`背包与图鉴` 分组包含 `神通`、`神通图鉴`、`炼器图鉴`、`仙种图鉴` 和 `帝兵图鉴`，玩家可以直接从菜单看到神通、炼器和高阶唯一道具成长线；`交易与资源处理` 分组包含万宝楼公开寄售入口。

同时保留 `menu_data`，用于兼容仍读取旧字段的菜单插件或旧配置。

## 安装

使用 NB-CLI 安装：

```bash
nb plugin install nonebot-plugin-xiuxian-signin
```

使用 pip 安装：

```bash
pip install nonebot-plugin-xiuxian-signin
```

如果暂未发布到 PyPI，也可以直接从 GitHub 安装：

```bash
pip install git+https://github.com/hszxjs/nonebot-plugin-xiuxian-signin.git
```


## 加载

在 NoneBot 项目中加载：

```python
nonebot.load_plugin("nonebot_plugin_xiuxian_signin")
```

或者在 `pyproject.toml` 中加入：

```toml
[tool.nonebot]
plugins = ["nonebot_plugin_xiuxian_signin"]
```

当前仓库的 `pyproject.toml` 已经加入此插件。

## 配置

可选配置项：

```env
XIUXIAN_SIGNIN_DATA_DIR=
XIUXIAN_SIGNIN_TIMEZONE=Asia/Shanghai
XIUXIAN_SIGNIN_IMAGE_WIDTH=900
XIUXIAN_SIGNIN_AVATAR_TIMEOUT=8
XIUXIAN_SIGNIN_FONT_PATH=
XIUXIAN_SIGNIN_BOLD_FONT_PATH=
```

| 配置项 | 默认值 | 说明 |
| --- | ---: | --- |
| `xiuxian_signin_data_dir` | localstore 插件数据目录 | 用户数据和群日榜数据保存目录；配置后会覆盖默认目录 |
| `xiuxian_signin_timezone` | `Asia/Shanghai` | 每日签到和 22:00 榜单结算的时区 |
| `xiuxian_signin_image_width` | `900` | 签到和垂钓图片宽度 |
| `xiuxian_signin_avatar_timeout` | `8.0` | QQ 头像下载超时时间 |
| `xiuxian_signin_font_path` | 内置字体优先 | 普通中文字体路径，通常无需配置 |
| `xiuxian_signin_bold_font_path` | 内置字体优先 | 粗体中文字体路径，通常无需配置 |

## 数据

用户数据默认保存到 `nonebot-plugin-localstore` 分配的插件数据目录：

```text
<localstore>/nonebot_plugin_xiuxian_signin/users.json
```

群每日话痨榜数据默认保存到：

```text
<localstore>/nonebot_plugin_xiuxian_signin/daily_chat_rank.json
```

群成员索引数据默认保存到：

```text
<localstore>/nonebot_plugin_xiuxian_signin/group_members.json
```

唯一帝兵、玩家交易和秘境救援委托分别保存到：

```text
<localstore>/nonebot_plugin_xiuxian_signin/unique_artifacts.json
<localstore>/nonebot_plugin_xiuxian_signin/trade_offers.json
<localstore>/nonebot_plugin_xiuxian_signin/mystic_rescue.json
```

如果需要迁移旧数据或指定固定目录，可配置 `XIUXIAN_SIGNIN_DATA_DIR` 指向可写目录。

插件已内置 `assets/fonts/NotoSansSC-VF.ttf`，默认优先使用内置字体。如果你想替换成自己的字体，可以手动配置字体路径，例如：

```env
XIUXIAN_SIGNIN_FONT_PATH=/path/to/your-font.ttf
XIUXIAN_SIGNIN_BOLD_FONT_PATH=/path/to/your-bold-font.ttf
```
